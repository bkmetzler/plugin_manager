# Plugin Manager (plugin-manager)

A lightweight plugin manager.


## Usage

```
    from plugin_manager.plugin_manager import PluginManager
    from pathlib import Path
    p = Path("example_plugins/")
    pm = PluginManager(p)
```


```
    from plugin_manager.plugin_manager import PluginManager
    pm = PluginManager("example_plugins/")
```
